# Skill Marketplace

A personal, Git-tracked marketplace for reusable Codex and Claude Code skills.

## Layout

- `SKILLS.md` - canonical skill index
- `skills/` - reusable skill packages
- `tools/` - supporting utilities such as stylometry

This repo is the source of truth. Clone it on each machine, sync changes with Git, and install or symlink the skills locally for the agent you are using.
